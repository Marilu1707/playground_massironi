María Luján Massironi

Estudiante de Negocios Digitales — UADE

Soy estudiante de la carrera de Negocios Digitales en UADE, con formación en marketing digital, análisis de datos y desarrollo web. Mi experiencia como Asistente Virtual me enseñó a combinar eficiencia operativa con pensamiento estratégico para resolver problemas reales.

Me interesa construir soluciones tecnológicas que sean funcionales, bonitas y fáciles de usar. Nido Mozzarella nació como un proyecto de portfolio para demostrar mis habilidades en desarrollo web con Django, combinando una identidad visual kawaii con funcionalidades reales como gestión de inventario, blog y sistema de pedidos.

🛠 Habilidades

Python Django SQL React.js Análisis de Datos Marketing Digital Gestión de Proyectos

📌 Sobre este proyecto

Nido Mozzarella es una aplicación web Django con temática de pizzería kawaii. Permite a los usuarios registrarse, explorar recetas del blog, gestionar un inventario de ingredientes y completar pedidos para ganar puntos. El panel de administración permite a los chefs (administradores) publicar y editar contenido.
